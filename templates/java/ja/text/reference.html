この付録では、1ページにKarelプログラミング言語の構造を定義します。

Base Karelはこう主張している。

move();

putBeeper();
pickBeeper();

カレルプログラムの構造：

/* コメントはどの部分にも含めることができます
 * の a プログラム. 彼ら 開始 と a スラッシュ-星
 * and end with a star-slash.
 */
 public class 名 extends スーパーカレル {

   public void run() {
       実行するコード
   }

    プライベートメソッドの宣言
}

}
